# teideninfo
Scraping script &amp; map viz for http://teideninfo.tepco.co.jp

# install and build

```
$ git clone https://github.com/frogcat/teideninfo.git
$ cd teideninfo
$ npm install
$ npm run geom
$ npm run data
```

- `npm run geom` で `docs/geom.json` が作成または更新されます
- `npm run data` で `docs/data.json` が作成または更新されます
